# Smart Building Energy Consumption Dashboard

An Excel-based energy analytics model for smart building data.

## What's Inside
| Sheet | Description |
|-------|-------------|
| BuildingData_Clean | Cleaned hourly energy data with engineered features |
| Descriptive Stat_Model | Descriptive statistics + regression output |
| Pivot Table | Average energy by building |
| Dashboard | Visual KPI dashboard |

## Features Analysed
- Energy consumption (kWh) by building, room, hour, and shift
- Occupancy rate vs energy usage
- Temperature bin analysis
- Multiple regression model (R² = 0.88)

## Tools Used
- Microsoft Excel (Power Query, Pivot Tables, Data Analysis ToolPak)
